## 🎮 操作指南

### 3D 视图操作

| 操作 | 方法 | 效果 |
|------|------|------|
| **旋转视角** | 按住鼠标左键拖动 | 360° 查看厂区布局 |
| **缩放** | 滚动鼠标滚轮 | 放大/缩小视图 |
| **平移** | 按住 Shift + 左键拖动 | 上下左右移动视角 |
| **悬停查看** | 鼠标悬停在设备上 | 显示单元名称和排放数据 |

### 侧边面板操作

| 功能 | 说明 |
|------|------|
| **实时排放监控** | 查看所有单元的排放数据表格 |
| **单元详情** | 下拉选择单元，查看详细参数 |
| **总排放统计** | 显示厂区总排放量和分布 |

## 🔍 各区域说明

### 预处理区（上方，蓝色系）
- 粗格栅 → 提升泵房 → 细格栅 → 曝气沉砂池 → 膜格栅
- 功能：去除大颗粒、砂粒，提升水位

### 生物处理区（中间，绿色系）
- 厌氧池 → 缺氧池 → 好氧池 → MBR膜池
- 功能：有机物降解、脱氮除磷

### 深度处理区（下方偏右，红色系）
- DF系统 → 催化氧化 → 消毒接触池
- 功能：深度过滤、高级氧化、消毒

### 污泥处理区（左侧，橙色系）
- 污泥处理车间
- 功能：污泥浓缩、脱水、干化

### 辅助设施
- 鼓风机房（为好氧池供氧）
- 除臭系统（收集处理臭气）

## 🆘 常见问题

**Q: 页面显示"ModuleNotFoundError"？**
A: 请运行 `pip install streamlit plotly pandas numpy`

**Q: 3D 场景显示为空白？**
A: 尝试刷新页面，或检查浏览器是否支持 WebGL

**Q: 鼠标操作无效？**
A: 确保点击了3D场景区域，再尝试拖动

**Q: 颜色没有变化？**
A: 在"工艺流程仿真"选项卡中修改参数后，需要切回3D选项卡刷新
