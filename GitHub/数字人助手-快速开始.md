# 3D数字人助手 - 快速开始指南

## 🎯 功能亮点

✅ **3D数字人展示**：可爱的「烷仔」形象，带呼吸动画和光效  
✅ **智能对话**：基于Coze平台的AI智能体，支持实时问答  
✅ **模拟模式**：无需配置即可体验（默认启用）  
✅ **历史记录**：自动保存对话，支持导出  
✅ **独立页面**：不影响原有任何功能  

## 📂 新增文件清单

| 文件 | 说明 |
|------|------|
| `coze_api.py` | Coze智能体API接入模块 |
| `digital_human_agent.py` | 3D数字人助手模块 |
| `assets/digital_human.jpg` | 数字人图片（烷仔） |
| `数字人助手配置说明.md` | 详细配置文档 |

## 🚀 快速启动

### 1. 安装依赖

```bash
pip install requests>=2.31.0
```

或一次性安装全部依赖：

```bash
pip install -r requirements.txt
```

### 2. 启动系统

```bash
streamlit run app.py
```

### 3. 使用数字人助手

1. 在浏览器中打开系统（默认 http://localhost:8501）
2. 点击顶部标签栏的「🤖数字人助手」
3. 在右侧聊天框输入问题，或点击快捷问题开始对话

## 💬 支持的问答主题

### 模拟模式（默认）
- 水厂工艺流程介绍
- 甲烷排放计算方法
- 工艺优化建议
- LSTM预测功能说明
- 减排技术对比

### Coze智能体模式（需配置）
- 更智能的对话体验
- 支持自定义知识库
- 更自然的语言理解

## 🔌 接入Coze智能体（可选）

### 步骤1：创建智能体
1. 访问 https://www.coze.cn
2. 创建智能体，配置人设和知识库
3. 发布并选择API方式

### 步骤2：获取凭证
- **Bot ID**：智能体详情页获取
- **API密钥**：开发者令牌中创建

### 步骤3：系统配置
1. 进入「🤖数字人助手」页面
2. 关闭「使用模拟模式」开关
3. 填入API密钥和Bot ID
4. 点击「验证API凭证」
5. 验证通过后即可使用

## 🎨 自定义设置

### 更换数字人形象
将新图片替换 `assets/digital_human.jpg`，保持相同文件名即可。

### 修改模拟回答
编辑 `coze_api.py` 中的 `MockCozeAPI.responses` 字典。

## 📞 常见问题

**Q: 数字人图片不显示怎么办？**  
A: 检查 `assets/digital_human.jpg` 是否存在，文件大小是否正常。

**Q: 如何清空对话历史？**  
A: 点击侧边栏的「🗑️ 清空当前对话」按钮。

**Q: API连接失败？**  
A: 检查网络连接，确认API密钥和Bot ID正确，智能体已发布。

**Q: 可以同时在其他页面显示数字人吗？**  
A: 目前采用独立页面方式，可在任何页面点击「🤖数字人助手」标签访问。
