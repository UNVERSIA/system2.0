# AI实验室 · 污水处理闯关小游戏 使用说明

## 📋 功能简介

AI实验室·污水处理闯关小游戏是一款互动式教育游戏，通过闯关的方式帮助用户学习和掌握污水处理的核心工艺流程。

## 🎯 游戏特点

### 1. 三个精心设计的关卡

| 关卡 | 名称 | 内容 | 工艺模块数量 |
|------|------|------|-------------|
| 第1关 | 🔧 预处理工艺 | 粗格栅→进水泵房→细格栅→曝气沉砂池→初沉池 | 5个 |
| 第2关 | 🦠 生物处理工艺 | 膜格栅间→厌氧池→缺氧池→好氧池→MBR膜池 | 5个 |
| 第3关 | ✨ 深度处理与出水 | 纤维滤池→臭氧接触池→紫外线消毒→出水泵房 | 4个 |

### 2. 核心功能

- **互动式学习**：点击工艺模块，按正确顺序排列
- **实时反馈**：即时检查答案，提供详细错误提示
- **工艺流程说明**：每关都有详细的工艺步骤讲解
- **科普小知识**：随机展示污水处理相关知识
- **通关系统**：记录通关进度和得分

### 3. 计分系统

- 基础分：100分/关
- 错误惩罚：-10分/次
- 提示惩罚：-5分/次

## 🎮 操作指南

### 开始游戏

1. 启动Streamlit应用：`streamlit run app.py`
2. 在浏览器中打开应用
3. 点击顶部选项卡 "🎮AI实验室·污水处理闯关"

### 游戏流程

1. **选择关卡**：点击关卡按钮选择要挑战的关卡
2. **排列工艺**：从左侧"可用工艺模块"区域点击模块，添加到右侧流程中
3. **调整顺序**：如需调整，点击已选模块旁的"❌"按钮移除重新排列
4. **检查答案**：点击"🔍 检查答案"按钮验证
5. **通关奖励**：答对后显示得分并解锁下一关

### 辅助功能

- **💡 查看工艺提示**：展开查看当前关卡的工艺提示
- **📚 工艺流程详细说明**：查看每个工艺步骤的详细介绍
- **💡 显示答案**：遇到困难时可查看正确答案（会扣分）
- **🗑️ 清空重排**：清空当前排列重新开始
- **🎓 污水处理科普小知识**：随机显示科普知识

## 🆘 故障排除

### 游戏无法加载

**现象**：点击游戏选项卡后显示错误信息

**解决方案**：
1. 检查 `water_treatment_game.py` 文件是否存在
2. 检查文件编码是否为 UTF-8
3. 检查Python版本是否 >= 3.8
4. 重新启动 Streamlit 应用

### 界面显示异常

**现象**：界面布局混乱或文字显示不全

**解决方案**：
1. 调整浏览器窗口大小
2. 使用Chrome或Edge浏览器
3. 清除浏览器缓存后刷新页面
