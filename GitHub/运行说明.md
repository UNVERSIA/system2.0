# 污水处理甲烷监测调控与智慧科普系统 - 运行说明

## 🚀 快速开始

### 1. 安装依赖

```bash
# 进入项目目录
cd system-main

# 安装所有依赖（包括tensorflow）
pip install -r requirements.txt
```

**注意**：如果tensorflow安装较慢或失败，可以使用国内镜像：
```bash
pip install tensorflow -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 运行系统

```bash
# 运行Streamlit应用
streamlit run app.py
```

然后在浏览器中打开显示的地址（通常是 http://localhost:8501）

## 📋 关于3D可视化

### ❌ 不要直接打开HTML文件

**旧的3D预览HTML文件已被删除**，因为它们直接双击打开会导致：
- 布局混乱
- JavaScript错误
- 无法正确显示3D场景

### ✅ 正确的查看方式

3D可视化功能已集成到Streamlit应用中：

1. 运行 `streamlit run app.py`
2. 在浏览器中打开应用
3. 点击第一个选项卡 **"3D水厂仿真"**
4. 即可看到完整的3D虚拟仿真场景

### 🎮 3D场景操作指南

| 操作 | 方法 |
|------|------|
| 旋转视角 | 按住鼠标左键拖动 |
| 缩放视图 | 滚动鼠标滚轮 |
| 平移场景 | Shift + 鼠标左键拖动 |
| 查看详情 | 鼠标悬停在设备上 |

## 🔧 系统功能

系统包含以下功能模块：

1. **3D水厂仿真** - 真实感3D虚拟仿真
2. **工艺流程仿真** - 2D工艺流程图
3. **甲烷足迹追踪** - 碳排放监测
4. **甲烷账户管理** - 碳账户统计
5. **优化与决策** - 减排方案优化
6. **甲烷排放预测** - LSTM预测模型
7. **减排技术分析** - 技术对比分析
8. **因子库管理** - 排放因子管理

## 💡 常见问题

### Q: 提示 "No module named 'tensorflow'"
**A**: 运行 `pip install tensorflow` 安装tensorflow库

### Q: 3D画面卡顿
**A**: 3D渲染需要一定的显卡性能，可以尝试：
- 关闭其他占用显存的程序
- 降低浏览器缩放比例
- 在侧边栏关闭"自动旋转"

### Q: 如何导出3D场景
**A**: 在Plotly图表右上角点击相机图标可以保存为PNG图片
