# 污水处理智能助手配置说明

## 📋 功能概述

本系统新增了【智能对话答疑】功能，包含：
- **智能助手展示**：可爱的卡通形象「烷仔」，带有呼吸动画和悬浮效果
- **智能对话**：基于Coze扣子平台的智能体，支持实时对话和答疑
- **历史记录**：自动保存对话历史，支持导出

## 📁 文件结构

## 🚀 使用方式

### 方式一：模拟模式（无需配置，立即可用）

默认启用模拟模式，智能体将使用预设的回答模板：

1. 启动系统：`streamlit run app.py`
2. 点击顶部的「🤖数字人助手」标签页
3. 直接开始对话

模拟模式支持的问答主题：
- 水厂工艺流程
- 甲烷排放计算
- 工艺优化建议
- LSTM预测使用
- 减排技术对比

### 方式二：接入真实Coze智能体（推荐）

#### 步骤1：在Coze平台创建智能体

1. 访问 [Coze扣子平台](https://www.coze.cn)
2. 登录并创建新智能体
3. 配置智能体：
   - 名称：污水处理助手 / 烷仔
   - 人设与回复逻辑：参考下方提示词
   - 添加知识库（可选）：上传水厂相关资料

#### 步骤2：发布智能体并获取API信息

1. 点击「发布」按钮
2. 选择「API」发布方式
3. 发布后获取：
   - **Bot ID**：在智能体设置页面查看
   - **API密钥**：在「开发者令牌」中创建

#### 步骤3：配置系统

在数字人助手页面的侧边栏中：

1. 关闭「使用模拟模式」开关
2. 输入Coze API密钥
3. 输入Bot ID
4. 点击「验证API凭证」
5. 验证成功后即可开始真实对话

## 🎨 自定义数字人形象

如需更换数字人形象：

1. 准备一张正方形的卡通形象图片（建议300x400像素）
2. 替换 `assets/digital_human.jpg`
3. 重启系统即可生效

## 📊 对话历史管理

- **自动保存**：所有对话自动保存到 `chat_history.json`
- **清空对话**：点击「🗑️ 清空当前对话」按钮
- **导出历史**：点击「📥 导出历史对话」按钮，生成JSON文件

## ⚙️ 高级配置

```env

```

编辑 `coze_api.py` 中的 `MockCozeAPI.responses` 字典，添加自定义问答：

```python
self.responses = {
    "你的问题": "你的回答",
    # ...
}
```

## 🐛 故障排除

### 问题1：数字人不显示
- 检查 `assets/digital_human.jpg` 是否存在
- 检查浏览器控制台是否有错误

### 问题2：API连接失败
- 确认API密钥和Bot ID正确
- 检查网络连接
- 确认智能体已发布且API已启用

### 问题3：对话无响应
- 检查是否处于模拟模式
- 查看Streamlit控制台错误信息
- 尝试清空对话历史
